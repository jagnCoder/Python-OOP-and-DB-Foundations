# Python Core Concepts & Database Foundations Lab 🚀

A comprehensive collection of mini-projects and scripts I built during my academic journey to master Core Python, Object-Oriented Programming (OOP), and Database Connectivity. 

I created this repository to document my learning progression from basic class structures to complex multi-threading and database-integrated management systems.

## 📂 Repository Structure

* **`Classes_and_Objects/`**: Foundational concepts of state and behavior.
* **`Inheritance_Polymorphism_and_Operator_Overloading/`**: Scripts demonstrating code reusability and method overriding.
* **`Iterators_Generators_Closures_and_Decorators/`**: Advanced Python features for memory efficiency and function wrapping.
* **`MultiThreading/`**: Practical implementations of concurrent execution (e.g., Grade Calculator, Spawning Threads).
* **`Regular_Expressions_and_Exception_Handling/`**: Input validation and custom error handling modules.
* **`Database_Connectivity/`**: **(Highlight)** Fully functional console applications using SQLite/MySQL, including:
  * ✈️ Flight Management System
  * 🏨 Hotel Management System
  * 🚂 Train Details Retrieval
  * 🏦 Bank Account Balance Updater

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Database:** SQLite / MySQL
* **Key Concepts:** OOP, Multi-threading, RegEx, CRUD Operations.

## 💡 What I Learned
Building these scripts solidified my understanding of how data flows from a Python backend to a database, and how to write clean, modular, and error-proof code.
